# 风控规则说明

## 风控级别

### P0 — 紧急

**触发条件（满足任一）：**
- 5分钟价格变化 > 3%
- 盘口深度下降 > 50%
- 巨鲸评分 > 85
- 闪崩/暴拉信号 = true

**执行动作：**
- 最大仓位：3%
- 分3批执行，间隔10秒
- 冷却时间：300秒
- 最大滑点：0.5%
- 全渠道通知
- 可覆盖冷却（双重审计）

### P1 — 升级

**触发条件（满足任一）：**
- 结构突破信号
- ATR > 1.5倍平均
- 巨鲸评分 60-85
- 状态切换信号

**执行动作：**
- 最大仓位：5%
- 分2批执行
- 冷却时间：120秒
- 最大滑点：0.3%

### P2 — 常规

**触发条件：**
- 趋势信号
- 巨鲸评分 < 60

**执行动作：**
- 最大仓位：8%
- 单批执行
- 冷却时间：60秒
- 最大滑点：0.2%

## 巨鲸覆盖规则

- whale_score > 85 且方向一致 → 升级P0，置信度+0.1
- whale_score > 85 且方向矛盾 → 降级hold，标记conflict
- whale_score > 75 → 风险降一级

## 系统降级

- score ≥ 80 → 正常运行
- score 50-80 → 暂停自动执行，仅通知
- score < 50 → 暂停所有操作，紧急告警
